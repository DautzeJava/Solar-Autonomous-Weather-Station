# Solar-Autonomous-Weather-Station
All the code related to the SAWS project, based on Arduino environement

SAWS is a Solar Autonomous Weather Station, based on an Arudino Pro Mini microcontroller and developed on the Arduino IDE.
It includes severals componant listed below:
-Arduino Pro Mini 5V
-Breadboard
-BME 680 (Weather sensor)
-GUVA S12SD (UV sensor)
-B10K
-TP4056
-MT3608
-Li-ion battery from JBL WAVE BEAM case
-LCD screen : 1602A
-JOYSTICK
-Resistors 
-Solar panel (3W 6V)
-A plastiv case

A video of demonstration of the project is currently in development.

This project was based on the Arduino IDE in C++ (applied to Arduino).
